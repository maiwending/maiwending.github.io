summary: "Prism DGTD simulation toolkit with GDM for pixelized metasurface This simulation package is the prism-based Discontinuous Galerkin Time Domain method with General Dispersion Model for analysis of gold pixelized metasurfaces. How to use it: Open ''runme.m'' in MATLAB. Modify the time span and the Geo matrix ''A''. The geometry is a 2-fold symmetric unit cell with a period of 400 nm. The unit cell can be patterned by several gold pixels, measures 50 nm × 50 nm × 20 nm. In order to add gold padding, one could input the number of padding, and then the index numbering."
